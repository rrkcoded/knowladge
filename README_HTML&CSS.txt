Element Label
Элемент label позволяет разместить текстовое описание элемента формы, которое связано с элементом input
Вложить input в label !


Attribute required
<input type="" required>
Атрибут required устанавливает поле формы обязательным для заполнения перед отправкой данных


Element Button
<button type=""> TEXT TEXT </button>
Элемент button представляет собой элемент управления , позволяющий пользователю запускать определенные действия .
Типы кнопок:
- submit – кнопка отправки формы. Так же это значение по умолчанию
- reset – кнопка, которая очищает введенные значения в форме
- button – кнопка которая не запускает никакое действие

Element input
<input type=""><input>
Элемент input позволяет создавать различные элементы формы .
Для этого у элемента существует атрибут type.


Attribute placeholder
<input type="email" placeholder="Ur mail">
Атрибут placeholder предназначен для вывода текста внутри поля формы, чтобы потом помочь пользователю ввести данные.


Attribute textarea
<textarea></textarea>
Элемент textarea представляет собой элемент формы для создания области, в которую можно вводить несколько строк текста


Property Box-sizing
Свойство box-sizing изменяет алгоритм расчета ширины и высоты элемента.
Свойство имеет несколько значений:
- content-box – при этом значение браузер рассчитывает размеры элемента по стандартному алгоритму
- border-box – width + height будут задавать ширину и высоту элемента, а не конкретной области


Property display
C помощью свойства display браузер понимает, как ему отобразить элемент и какими свойствами элемент обладает


Правила синтаксиса при названия классов:
- Название должно передавать смысл элемента
- Быть на англ языке
- В название только строчные буквы
- Для названий из более чем 1 слова используем camelCase

position:
- static (default)
- relative Когда position установлено относительно, элемент может перемещаться относительно его текущей позиции.
- fixed Когда position задано как fixed, оно действует как абсолютное: вы можете установить координаты слева/справа и сверху/снизу.
- absolute Когда position устанавливается абсолютно, элемент может перемещаться относительно первого позиционированного предка.
